
# Pomodoro Timer (проект для Агоны)

Мобильное приложение для организации учебного времени, совместной концентрации и отслеживания продуктивности в формате Pomodoro. Проект вдохновлён функциональностью YPT Pomodoro Timer.

## О проекте

Pomodoro Timer помогает пользователям планировать учебные сессии, запускать таймер концентрации и отслеживать собственную активность. Приложение поддерживает работу с учебными комнатами и группами, где пользователь может видеть время занятий других участников и сравнивать свою продуктивность с прогрессом группы.

Основная логика приложения построена вокруг Pomodoro-сессий. Пользователь может настроить длительность фокус-сессии и перерыва, запустить таймер, поставить его на паузу или завершить занятие. После завершения сессии данные сохраняются и используются для отображения истории и статистики.

Отдельное внимание уделено групповому формату: пользователь может вступать в комнаты, отслеживать активность других участников и видеть общее время обучения внутри группы. Такой подход помогает поддерживать мотивацию и формировать привычку регулярных занятий.

Проект реализован с разделением на слои, локальным хранением данных и реактивным обновлением интерфейса. Часть ключевой бизнес-логики будет покрыта unit-тестами.

## Основные функции

* Запуск, пауза и сброс Pomodoro-таймера
* Настройка длительности фокус-сессии и перерыва
* Создание и просмотр учебных комнат
* Работа с группами пользователей
* Отображение времени занятий других участников
* Сохранение завершённых учебных сессий
* Просмотр истории занятий
* Отображение статистики продуктивности
* Локальное хранение пользовательских данных
* Реактивное обновление UI при изменении состояния приложения

## Технологии

* Kotlin

* Kotlin Coroutines, Flow
